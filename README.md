# Beadbox Homebrew Cask

Homebrew cask for [Beadbox](https://github.com/beadbox/beadbox), the macOS app for the beads issue tracker.

## Install

```bash
brew tap beadbox/cask
brew install --cask beadbox
```

## Update

```bash
brew upgrade --cask beadbox
```

## Uninstall

```bash
brew uninstall --cask beadbox
```
